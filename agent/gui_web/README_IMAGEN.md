# Captura de pantalla de la interfaz F3-OS Assistant

La imagen de ejemplo de la interfaz debería guardarse aquí como `screenshot.png`

Para capturar la interfaz:
1. Inicia el servidor: `cd agent && ./run.sh gui-server`
2. Abre `http://localhost:8080` en tu navegador
3. Captura la pantalla
4. Guarda la imagen como `agent/gui_web/screenshot.png`

La interfaz muestra:
- Indicador de vida del agente (arriba a la derecha)
- Panel de estado con métricas
- Área de chat interactivo
- Diseño futurista estilo Star Wars/F1
